# Pucks With Purpose Hockey Website

This is a static website. It does not use npm, Next.js, React, or any build dependencies.

## Vercel setup

When importing this project into Vercel:

- Framework Preset: Other
- Build Command: leave blank
- Output Directory: leave blank or `.`
- Install Command: leave blank

Make sure these files are at the top/root level:

- index.html
- styles.css
- vercel.json

## Domain

After deployment, add `puckswithpurposehockey.com` inside Vercel Project Settings > Domains. Vercel will tell you the exact GoDaddy DNS records to add.
